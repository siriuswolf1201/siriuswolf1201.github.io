# assets/ — 圖檔插槽

放官方圖檔（可選），網頁會自動取代內建的文字後備：

| 檔名 | 用途 | 沒放會怎樣 |
| --- | --- | --- |
| `logo.png` | 左上角 Rotaract District 3482 標誌 | 用文字「Rotaract / District 3482」後備 |

> 想 100% 還原海報大標字，可放一張去背的「FM348.2」標題 PNG，
> 再到 `index.html` 把 `<h1 class="brush-title">FM348.2</h1>` 換成 `<img>` 即可。
> 目前預設用 Google Fonts 的 Permanent Marker 筆刷字模擬。
