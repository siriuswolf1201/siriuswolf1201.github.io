# audio/ — 各社語音檔放這裡

把每個社團的介紹語音（建議 **mp3**，也支援 m4a/ogg/wav）放進這個資料夾，
然後到 [`../data/stations.js`](../data/stations.js) 把對應社團的 `audio` 欄位填上路徑：

```js
{ id: 1, name: "○○扶輪社", freq: 340.2, audio: "audio/club-01.mp3", intro: "…", color: "#f4a259" },
```

## 注意事項
- **檔名請用英文/數字**（例如 `club-01.mp3`），避免中文檔名在某些主機上載入失敗。
- `audio` 留**空字串** `""` → 該台會自動播放合成佔位音，畫面照常運作（方便先測試）。
- 也可以放**外部連結**（完整 https 網址，例如雲端直連檔），但對方需允許跨來源播放。
- 檔案太大會影響載入；建議單檔壓到合理大小（語音用 ~64–96kbps 即可）。
